The Schedule is GenreTV's public view of release-relevant Seasons. It is not a complete season history: older Seasons remain available in management while the Schedule shows the rows useful for deciding what is on, coming, unresolved, or finished.

![The GenreTV Schedule with four release sections, filters, and the Now Showing table.](/docs/screenshots/schedule.png)

## Read a row

The desktop table presents:

- **Show**: the Show title, linked to IMDb when available;
- **Status**: useful lifecycle or release context;
- **Where**: ordered networks, streamers, or other Organizations, linked to official sites when available;
- **Genre**: descriptive Genre Tags;
- **Seasons**: the Official Season Count, with Extra Releases shown separately where relevant;
- **When**: the release date or Release Window; and
- **Finale**: the known finale timing when applicable.

Select **+** to reveal links, notes, and other available details. Selecting a Genre value applies it as a filter.

## Mobile view

On a narrow screen, each row keeps the Show title and timing visible. Secondary values such as platform, Genre, language, and season count move into the expandable details. The section tabs use an icon and row count so all four remain reachable without horizontal scrolling.

## Counts and pagination

Each section tab shows its current row count. The footer reports the filtered row total and page number. Use **Rows** to display 20, 50, or 100 results per page.

The same Show can have more than one upcoming row when multiple future Seasons or Extra Releases are known.
