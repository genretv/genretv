---
title: Search, filter, and sort
description: Narrow a schedule section without changing the underlying list.
---

Controls apply to the active Schedule Section. Their available Language, Genre, and platform values are derived from rows in that section, so switching sections can change the choices.

## Search

Enter text in **Search** to narrow the visible rows. Search is useful for titles and the metadata included by the current schedule query.

## Filters

- **Platform** selects an Organization such as a network or streamer.
- **Language** accepts zero or more language codes.
- **Genre** accepts zero or more Genre Tags.

Select a Genre directly in a table row to apply it. Open a multi-select filter and choose **Clear** to remove all values.

## Sorting

Use **Sort** or a sortable table heading to choose the ordering. Selecting the same heading again reverses its direction. Finished starts with newest timing first; the other sections start with earliest timing first.

## Browser-local preferences

GenreTV saves these viewing preferences in local browser storage for both anonymous and signed-in visitors. They create free filtered views of the same list; they do not create separate lists or publish changes.

HTML exports ignore search, filters, sorting, pagination, and expanded-row state. They always contain all four sections in default order.
